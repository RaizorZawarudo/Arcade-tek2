games:
- Nibbler
- PacMan

graphical libraries:
- ncurses
- SDL2
- SFML

Arcade Machine Inputs:
  - i: game selection up
  - k: game selection down
  - j / a: graphics selection left
  - l: graphics selection right
  - q: quit program
  - r: reload list of libs
  - space: launch selected game
